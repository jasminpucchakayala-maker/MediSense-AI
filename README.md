# 🩺 MediSense AI

An AI-powered disease prediction web application built with Streamlit and Machine Learning.

## Features

- Disease Prediction
- PDF Health Reports
- Prediction History
- Analytics Dashboard
- Model Comparison
- Interactive UI

## Technologies

- Python
- Streamlit
- Scikit-learn
- Pandas
- ReportLab
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
