# 🏥 AI-Powered Disease Prediction & Clinical Decision Support System

**MediSense AI** is a modern healthcare application built using **Python** and **Streamlit** that leverages Machine Learning to predict diseases based on user-selected symptoms. The application provides an intuitive interface for disease prediction, analytics visualization, prediction history management, and professional PDF report generation.

> **⚠ Disclaimer:** This project is developed for educational and research purposes only. The predictions generated by this application should **not** be considered as a replacement for professional medical advice or diagnosis.

# 🚀 Overview

MediSense AI aims to demonstrate how Machine Learning can assist in preliminary healthcare analysis by identifying possible diseases based on patient symptoms.

The system provides:

- Intelligent Disease Prediction
- Interactive Analytics Dashboard
- Prediction History Tracking
- Professional Clinical PDF Reports
- Modern Healthcare User Interface

# ✨ Features

## 🩺 Disease Prediction

- Predict diseases using Machine Learning
- User-friendly symptom selection
- Instant prediction results
- Clean healthcare-inspired interface

Interactive dashboard displaying:

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/jasminpucchakayala-maker/MediSense-AI
```

### Navigate to project

```bash
cd MediSense-AI
```

### Create virtual environment

```bash
python -m venv .venv
```

### Activate environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Linux / macOS

```bash
source .venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶ Running the Application

```bash
streamlit run app.py
```

The application will open automatically in your browser.
